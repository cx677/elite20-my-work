# Artifact 生成记录 – 让 Claude Code 生成比赛计划

## 任务目标
作为课程 Day 5 的 artifact，我需要用 AI 辅助生成一份参加"教育数字人大赛"的详细计划。我本人并不一定真实参赛，重点在于展示"驾驭 AI 产出高质量计划"的能力。

## 使用的工具
- Claude Code（运行于本地，模型为 DeepSeek-V4-Pro）

## 我提供的 Prompt（完整复制）
```
请帮我制定一份详细可行的参赛计划，目标是参加第三届"教育数字人大赛——我和数字人的故事"（普通高等教育组）。我是 AI+X 精英班的大二学生，通信工程专业，完全是小白。

请基于以下比赛要求（我简要提供）：
- 主题：我和数字人的真实故事，强调人机协同、共同成长。
- 材料：申报书、演示文稿（PDF）、演示视频（≤10分钟，≤500MB）、可选支撑材料。
- 评分重点：教育价值（30%）、技术融合（25%）、共创机制与成长成效（25%）、反思深度（20%）。
- 截止时间：理论上2026年4月20日已过，但我作为练习，仍按标准流程做，产出完整作品用于课程项目或下次比赛。

我需要你为我生成一份详细的、小白能直接执行的计划，内容包括：
1. 建议我使用哪个免费/低门槛的数字人平台（如腾讯智影、D-ID、剪映等），并说明理由。
2. 一个真实、可操作的"故事选题"示例（最好与通信工程或通用学习痛点相关），包含：痛点、数字人角色、交互方式、预计成长证据。
3. 分步骤执行流程（从准备到最终提交），每步给出具体操作和所用时间。
4. 如何制作演示视频（脚本结构、拍摄要点、剪辑建议）。
5. 如何整理演示文稿（PPT结构建议）。
6. 如何填写申报书与准备支撑材料。
7. 如何记录"创作过程"（trace）以符合课程要求。

请以 Markdown 格式输出，语言口语化，步骤清晰，适合新手。同时，告诉我如果我没有技术基础，哪些地方可以简化或找替代方案。
```

## AI 的输出摘要
Claude Code 返回了一份约 400 行的完整 Markdown 参赛计划（保存为 `D5_competition_plan.md`），主要内容包括：

- **平台推荐**：首选剪映（CapCut）专业版，理由是免费、数字人生成与视频剪辑一体化、零门槛；备选腾讯智影和 D-ID/HeyGen。
- **故事选题示例**：《当通信生遇上数字人——我的"信号与系统"学习搭子》，设计了一个叫"小傅"的数字人学习搭子，通过"被动解惑 → 主动对话 → 反向输出"三阶段实现从期中 62 分到期末 82 分的成长。同时提供了 4 个备选选题方向。
- **5 周执行流程**：第 1 周选题与素材收集 → 第 2 周申报书与脚本 → 第 3 周素材制作 → 第 4 周剪辑整合 → 第 5 周最终提交，每步标注了具体操作和预估工时。
- **演示视频制作指南**：8 段脚本结构模板（开场→痛点→数字人诞生→三阶段→成长对比→反思），拍摄要点（收音>画面、灯光、稳定、构图），剪辑建议（节奏、转场、字幕校对、BGM），以及导出参数设置。
- **PPT 结构建议**：15 页详细页面规划，含配色与字体建议。
- **申报书撰写指南**：7 部分结构模板（概述→故事叙述→教育价值→技术方案→人机协同→反思展望→参考资料），含字数分配和写作要点。
- **创作过程记录（Trace）模板**：提供了 Markdown 格式的记录模板和记录要点。
- **小白避坑指南**：7 种常见困难的简化策略 + 5 个新手最容易犯的错误。
- **工具清单**：9 款免费工具的速查链接表。
